# Install: 
Install include running Front End project and Back End project. Please Install following the manual.
## BackEnd:
    - cd be
    - yarn 
    - cd be/app
    - yarn start  - with yarn/ npm start   - with npm
 ## Run Seeder:
    - cd be/app
    - npx sequelize-cli db:seed:all
 ## Front - End
    - cd FE/FoodApp
    - run cmd with Adminnistrator
    - expo start
    - Scan QR
